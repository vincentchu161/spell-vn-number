spell-vn-number

A lightweight JavaScript/TypeScript library for spelling out numbers in Vietnamese

Installation

npm install spell-vn-number

Kết quả (Result):

Các ví dụ (Examples):

123 1,234 10,001 100,000,000 -12,345.67 0.123 00.4050 00.798 00.000 1000000000000000
